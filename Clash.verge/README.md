## 使用方法

### 1. Merge 配置（可选，用于覆盖或补充特殊字段）
- 打开 Clash Verge 的配置管理界面。
- 点击「新建」，选择类型为 **Merge**。
- 将 `Merge` 文件夹中的内容复制粘贴到配置框中，然后保存。

### 2. Script 规则（用于自定义分流策略）
- 在配置管理界面点击「新建」，选择类型为 **Script**。
- `Script` 文件夹中提供两种规则格式，请**任选其一**使用：

  - **RuleSet**  
    - **优点**：规则高度透明、结构清晰，支持灵活组合与精细控制。
    - **适合人群**：希望完全掌控分流逻辑、或需要自定义特定规则的用户。
    - **注意**：你可以自行编辑或新增规则，实现个性化分流策略。

  - **GeoSite**
    - **优点**：开箱即用、配置简单，适合快速部署。
    - **局限性**：规则由预编译的域名列表构成，内容不透明，难以确认具体包含哪些站点；且更新依赖上游维护，可能存在覆盖不全的情况。
    - **适合人群**：追求便捷、不愿手动维护规则的用户。

- 选择你偏好的类型后，将对应文件的内容复制粘贴到配置框中并保存。

### 3. 应用自定义规则到机场订阅
- 导入或更新你的机场订阅链接后，Clash Verge 会自动应用你配置的 Merge 和 Script 规则，替代原始订阅中的默认规则。

### 4. 个性化调整
- 若当前配置不完全符合你的使用习惯，可随时修改规则内容，灵活适配你的网络环境与需求。
